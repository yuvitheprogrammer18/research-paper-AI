# 🔬 ResearchPaper AI Studio

ResearchPaper AI Studio is an AI-powered academic research assistant built with Streamlit and Gemini AI.

It helps students generate research topics, find academic references, analyze research papers, identify research gaps, build methodologies, create citations, and export complete research papers.

## Features

### 💡 Topic Generator

* Generate research topics
* Difficulty scoring
* Novelty scoring
* Research gap suggestions

### 📚 Reference Finder

* Search ArXiv papers
* Search CrossRef publications
* Build a personal reference library

### 📄 PDF Analyzer

* Upload research papers
* Extract text from PDFs
* Generate summaries
* Identify methodology and findings

### 📊 Literature Matrix

* Compare multiple papers
* Export comparison tables

### 🔍 Research Gap Finder

* Discover unexplored areas
* Generate research opportunities

### 🧪 Methodology Builder

* Research objectives
* Variables
* Data collection methods
* Evaluation metrics

### ✍️ Paper Builder

* Abstract generation
* Introduction
* Literature review
* Methodology
* Conclusion

### 🔖 Citation Manager

* APA
* MLA
* IEEE
* Chicago

### 📤 Export Options

* DOCX Export
* PDF Export

## Tech Stack

## Installation

```bash
git clone https://github.com/yourusername/ResearchPaper-AI-Studio.git

cd ResearchPaper-AI-Studio

pip install -r requirements.txt

streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```
